## Commands

### Build
- `[BUILD_COMMAND]`: Command to build the entire project.
- `[BUILD_SUBPROJECT_COMMAND]`: Command to build a specific subproject or module.

### Lint
- `[LINT_COMMAND]`: Command to run linting checks.
- `[FIX_LINT_COMMAND]`: Command to automatically fix linting issues.

### Test
- `[TEST_ALL_COMMAND]`: Command to run all tests.
- `[TEST_UNIT_COMMAND]`: Command to run unit tests.
- `[TEST_INTEGRATION_COMMAND]`: Command to run integration tests.
- `[TEST_SINGLE_FILE_COMMAND]`: Command to run tests for a specific file (e.g., `pytest path/to/test_file.py` or `npm test -- path/to/test_file.js`).

### Run
- `[RUN_DEV_COMMAND]`: Command to start the development server.
- `[RUN_PROD_COMMAND]`: Command to run the application in production mode.
